# Zayed Character Pack

Replaces Felix with a custom Zayed character featuring the Pearl Diver model.

## Installation

### Quick Install (Recommended)
```bash
python mod_installer.py install ZayedMod
```

### Manual Install
See `COMPLETE_CHARACTER_INJECTION_PROTOCOL.md` for manual steps.

## Uninstall

```bash
python mod_installer.py uninstall
```

This will restore all original game files from backup.

## Contents

- **3D Model:** Pearl Diver mesh with custom materials
- **Portrait:** Custom Zayed portrait (214x251)
- **Name:** "Zayed" replaces "Felix"

## Requirements

- Endstar game installed
- Python 3.x with Pillow (`pip install Pillow`)
- Original game files backed up (installer does this automatically)

## Important Notes

1. Launch `Endstar.exe` directly, NOT through the launcher
2. The launcher may restore original files
3. Always backup your game before modding
