Pearl Diver Character Mod for Endstar
=====================================

This mod replaces the Felix character with a custom Pearl Diver character.

Installation:
1. Run InstallMods.bat (requires Python 3)
2. Launch game with LaunchEndstar.bat or run Endstar.exe directly
   IMPORTANT: Do NOT use Endless Launcher!

Uninstall:
- Run UninstallMods.bat to restore original files

Known Issues:
- Character appears as silhouette in Party/lobby preview
- Works correctly during gameplay

Created with Meshy AI + Unity 2022.3.62f2
